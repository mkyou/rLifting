# General Thresholding Wrapper

General Thresholding Wrapper

## Usage

``` r
threshold(x, lambda, method = "soft")
```

## Arguments

- x:

  Input vector.

- lambda:

  Threshold value.

- method:

  Method: "hard", "soft" or "semisoft".

## Value

Numeric vector of the same length as `x` with thresholded coefficients.
