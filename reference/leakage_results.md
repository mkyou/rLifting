# Leakage (Impulse Response) Results

Measurement of energy leakage into the "past" when processing an impulse
signal. Used to demonstrate the zero-lookahead property of the causal
mode.

## Usage

``` r
data(leakage_results)
```

## Format

A data frame with:

- Method:

  Method description (e.g. "rLifting causal").

- Leakage:

  Sum of squared differences (leakage energy).
